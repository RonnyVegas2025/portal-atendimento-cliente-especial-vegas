# Portal Parceiros Vegas

Base inicial do projeto para usar com GitHub, Vercel e Supabase.

## Tecnologias
- Next.js 14
- React 18
- Tailwind CSS
- Framer Motion
- Lucide React
- Supabase

## Rodar localmente
```bash
npm install
npm run dev
```

Acesse:
`http://localhost:3000`

## Publicar na Vercel
1. Suba este projeto no GitHub
2. Entre na Vercel
3. Add New > Project
4. Selecione o repositório
5. Deploy

## Variáveis de ambiente
Crie `.env.local`:
```env
NEXT_PUBLIC_SUPABASE_URL=sua_url
NEXT_PUBLIC_SUPABASE_ANON_KEY=sua_chave
```
